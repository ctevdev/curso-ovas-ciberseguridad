SCORM INDIVIDUALES - CURSO DE CIBERSEGURIDAD
============================================

Esta carpeta contiene paquetes SCORM 1.2 independientes.

Cada archivo ZIP debe subirse a Moodle como una actividad SCORM separada.
No se debe subir una carpeta integrada del curso ni un ZIP que contenga varias
OVAs al mismo tiempo.

ESTRUCTURA

unidad-1/
- OVA-01-introduccion-ciberseguridad-SCORM12.zip
- OVA-02-cuentas-contrasenas-navegacion-SCORM12.zip
- OVA-03-dispositivos-wifi-nube-backup-SCORM12.zip
- OVA-04-laboratorio-phishing-fraudes-SCORM12.zip
- OVA-05-plan-personal-ciberhigiene-SCORM12.zip

unidad-2/
- OVA-U2-01-activos-personales-profesionales-SCORM12.zip

COMO USAR EN MOODLE

1. Entrar al curso en Moodle.
2. Activar edicion.
3. Seleccionar "Anadir una actividad o un recurso".
4. Elegir "Paquete SCORM".
5. Subir solo uno de los ZIP listados arriba.
6. Configurar calificacion maxima en 100.
7. Guardar la actividad.
8. Repetir el proceso para cada OVA.

IMPORTANTE

- Cada ZIP ya contiene su propio imsmanifest.xml en la raiz.
- Cada ZIP ya contiene su propio index.html en la raiz.
- Cada ZIP tiene sus propios archivos css, js y assets.
- Cada OVA reporta score.raw y lesson_status de forma independiente.
- No combine varias OVAs dentro de un solo paquete SCORM.
- No suba la carpeta curso-ovas-ciberseguridad como un unico SCORM.
- No suba la carpeta scorm-individuales como un unico SCORM.

VALIDACION RAPIDA

Un paquete individual correcto debe abrirse como ZIP y mostrar en la raiz:

- imsmanifest.xml
- index.html
- css/
- js/
- assets/

Si Moodle muestra varias OVAs dentro de una sola actividad, se subio el archivo
equivocado. Debe crearse una actividad SCORM por cada ZIP individual.
