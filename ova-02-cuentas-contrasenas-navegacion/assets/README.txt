OVA 2 - PROTECCIÓN DE CUENTAS, CONTRASEÑAS, CORREO, REDES SOCIALES Y NAVEGACIÓN

Esta carpeta es un paquete SCORM 1.2 independiente.

CÓMO COMPRIMIR

Entre en la carpeta ova-02-cuentas-contrasenas-navegacion y comprima:

- index.html
- imsmanifest.xml
- css
- js
- assets

imsmanifest.xml debe quedar en la raíz del ZIP.

CÓMO SUBIR A MOODLE

1. Añada una actividad "Paquete SCORM".
2. Suba el archivo ZIP.
3. Configure calificación máxima en 100.
4. Use "Calificación más alta" o "Último intento".
5. Active finalización por estado aprobado o completado.

DISTRIBUCIÓN DE PUNTAJE

- Caso inicial de Valentina: 25 puntos.
- Actividad de estructura de frase de contraseña: 20 puntos.
- Auditoría express de cuentas: 30 puntos.
- Evaluación final: 25 puntos.

La actividad "Detecta el error de seguridad" es complementaria y no afecta el
puntaje SCORM.

IDENTIDAD Y RECURSOS VISUALES

La OVA utiliza la identidad azul/cian del diplomado, con logo Universidad de
Cartagena + CTEV, fondos hexagonales y recursos visuales locales. No se cargan
imágenes desde internet.

RECURSOS EXTERNOS OPCIONALES

La OVA incluye enlaces de profundización que requieren internet. No son
obligatorios y no afectan el reporte SCORM:

- Google Cybersecurity Professional Certificate.
- Foundations of Cybersecurity - Google/Coursera.
- Google Security Checkup.
- Google 2-Step Verification.
- Google Password Manager.
- NIST SP 800-63B.

VALIDACIÓN SCORM

Al finalizar, revisar en Moodle:

- cmi.core.score.raw: puntaje entre 0 y 100.
- cmi.core.lesson_status: passed si el puntaje es 70 o más; completed si es
  menor a 70 pero completó actividades; incomplete si faltan actividades.
- cmi.suspend_data: progreso, respuestas y estado final.
